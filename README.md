# กาเนส พร้อมสอบ ป.1

เว็บแอปฝึกความพร้อมก่อนสอบเข้า ป.1 สำหรับน้องกาเนส ตามแนวทางจากไฟล์คู่มือผู้ปกครอง สวนสุนันทา ฝ่ายประถม

## สิ่งที่มีในแอป

- กิจกรรม 5 ฐาน: ฟังคำสั่ง, ภาษาไทย, คณิต/เชาวน์, ลากเส้น, สัมภาษณ์
- ปุ่มเสียงอ่านโจทย์ทุกกิจกรรม เหมาะกับเด็กที่ยังอ่านไม่คล่อง
- ปุ่มแตะขนาดใหญ่ ใช้งานบน iPad ได้ดี
- กระดานลากเส้นด้วยนิ้ว
- แผงผู้ปกครองพร้อมตารางฝึก 30 นาทีและเช็กลิสต์ก่อนสอบ
- PWA พร้อม manifest, service worker และไอคอนสำหรับติดตั้งบนหน้าจอ iPad

## ใช้งานในเครื่อง

```bash
npm install
npm run dev
```

## สร้างไฟล์เว็บ

```bash
npm run build
```

## GitHub Pages

repo นี้มี workflow สำหรับ deploy ไป GitHub Pages เมื่อ push เข้า branch `main`

หลัง deploy สำเร็จ เว็บจะอยู่ที่:

```text
https://basssg.github.io/GaNesh_suansunandha/
```
